# hcu-network
현대청운고등학교 졸업생이 [현대청운네트워크](hcuhs.kr)의 복각 사이트를 만들었습니다. 실제 사이트는 PHP 기반이지만, 이 사이트는 정적 사이트이기 때문에 HTML, CSS, 그리고 client-side JS만을 이용하여 재구성되었습니다. 고로 원본 사이트와 약간의 차이는 존재할 수 있으며, 당연히 '신청하기' 등과 같이 서버로 전송하는 기능은 포함되지 않았습니다.
